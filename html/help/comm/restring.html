Restring


Every shop offers restring for the objects it'd buy too - so you might have to look around a bit, which shop restrings which object.  There are 3 types of restring and normally it makes senes to change all three.

name => Keyword list for the object
short => short name, inventory name
long => the ind of text that is shown in room description

Every restring action costs 10000 gold.

Synatax:
  restring <type> <object> <new string>
  eg. restring name shoes shoes clown green
  eg. restring short shoes some green clown shoes
  eg. restring long shoes A clown has forgot his green shoes here.
