OOC Communication / Gossip / Herotalk / Tell / Ignore

This is an out of character communication command that broadcasts a message to all other players. There Is also a ranged version of tell which counts as OOC too. With the ignore command you can decide whose OOC tells to ignore. Ignore with an argument switches the ignore setting on/off.

Syntax:

  GOSSIP <text>
  GOSSIP #<social>
  alternatively use OOC or .
  HERO <text>
  HERO #<social>
  alternatively use +
  TELL <character> <text>
  IGNORE
  IGNORE <character>
