# Professions File
#
# This defines the professions a character can possess
#
# Format:
#
#  PROFESSION 'name' 		// Name of the profession
#
#	Desc			// Short description of the profession
#  description~			// 
#
#	Skill	name level	// Level when the skill is gained
#				//   0 means you start with it as a
#				//     base skill (value 42).
#
#	Kit	vnum		// The vnum of a piece of equipment
#				// that a character with the profession
#				// receives.  Given at initial outfit,
#				// outfit command and career change.
#				// Object will be modified to have value 0.
#				//
#				//	47	Passport
#				//	56	Mace
#				//	57	Dagger
#				//	58	Sword (bowie knife)
#				//	59	Vest
#				//	60	Shield
#				//	61	Lantern
#				//
#
#	Stats	str int wis dex con
#				// This lists the number of d4 to be 
#				// rolled to generate the stats for a
#				// character who begins play in this
#				// profession.  A full rool is 5 dice,
#				// and omitted dice are counted as 4s.
#				// At least one dice must be rolled.
#
#	Prime	stat		// This identifies the 'prime stat' for 
#				// the profession which is figured into
#				// some training and advancement rolls.
#				//	0	STR
#				//	1	INT
#				//	2	WIS
#				//	3	DEX
#				//	4	CON
#
#	PC	Yes | No	// Y means available to PCs, N otherwise.
#
#	Initial Yes | No	// Y means available to new PCs, N means
#				// only available to experienced PCs.
#
#	Start	vnum		// Vnum of the room where a member of the
#				// profession initially enters the mud.
#
#	Cond	condition	// This specifies a condition that must be
#				// stisfied before the profession can be
#				// learnt. Conditions are:
#				// 	1 'skill_name' low high
#				//	2 'level' low high
#				//	3 'race_name' 0 0
#				//	4 'prof_name' low high
#				//	5 'light' low high
#				//	6 'align' low high
#				//	7 'moon' low high
#				//	8 'clan_name' low high
#				//	9 'time' low high
#				//	10 'month' low high
#				//	11 'gender' g1 g2
#				//	19 'deed' deed 1 
#
#	ICond	condition	// Specifies a condition that has to be
#				// met for the character to be created
#				// in the profession.  Normally just using
#				// race, gender and alignment.
